cppе’ҢgolangиҜӯиЁҖеұӮйқўи®ҫи®Ўзҡ„еҢәеҲ«вҖ”жҰӮиҝ°

conceptual differences

  1. GO жІЎжңүеёҰжңүжһ„йҖ еҮҪж•°е’Ңжһҗжһ„еҮҪж•°зҡ„зұ»пјҢдёәдәҶжӣҝд»Јзұ»зҡ„ж–№жі•гҖҒзұ»зҡ„еӨҡйҮҚз»§жүҝгҖҒиҷҡеҮҪж•°пјҢGoжҸҗдҫӣдәҶжҺҘеҸЈгҖӮC++дёӯдҪҝз”Ёзұ»дјјдәҺGoзҡ„жҺҘеҸЈжҰӮеҝөзҡ„жҳҜtemplates
  2. GoжҸҗдҫӣиҮӘеҠЁеһғеңҫеӣһж”¶жңәеҲ¶пјҢдёҚйңҖиҰҒжҳҫзӨәең°йҮҠж”ҫеҶ…еӯҳз©әй—ҙгҖӮдёҚйңҖиҰҒжӢ…еҝғе Ҷе’Ңж Ҳзҡ„еҲҶй…ҚйқһйҮҠж”ҫпјҢжҜ”еҰӮC++дёӯжҸҗдҫӣзҡ„new malloc delete delete[] freeпјҢд№ҹдёҚз”ЁеҚ•зӢ¬ең°з®ЎзҗҶжҷәиғҪжҢҮй’ҲпјҲstd::unique_ptr, std::shared_ptr, std::weak_ptr, std::auto_ptrпјүд»ҘеҸҠеҺҹе§ӢжҢҮй’Ҳзұ»еһӢгҖӮGoзҡ„run-time systemеҸҜд»ҘеӨ„зҗҶеҘҪжүҖжңүзҡ„зЁӢеәҸе‘ҳе®№жҳ“зҠҜй”ҷзҡ„ең°ж–№
  3. GoжңүжҢҮй’ҲпјҢдҪҶжҳҜжІЎжңүpointer arithmetic гҖӮGoжҢҮй’ҲжӣҙеғҸC++зҡ„еј•з”ЁгҖӮдёҚиғҪеӨҹдҪҝз”ЁG0жҢҮй’ҲеҸҳйҮҸйҒҚеҺҶеӯ—з¬ҰдёІзҡ„жҜҸдёӘеӯ—иҠӮгҖӮGoдёӯзҡ„SlicesеҸҜд»Ҙж»Ўи¶іеӨ§йғЁеҲҶйңҖиҰҒжҢҮй’Ҳиҝҗз®—зҡ„жғ…жҷҜ
  4. Goй»ҳи®ӨжҳҜе®үе…Ёзҡ„гҖӮжҢҮй’ҲдёҚиғҪжҢҮеҗ‘д»»ж„Ҹзҡ„еҶ…еӯҳпјҢд»ҘеҸҠзј“еӯҳеҢәзӣҠеӨ„еҜјиҮҙзЁӢеәҸеҙ©жәғпјҢGoдёҚеӯҳеңЁиҝҷдәӣе®үе…Ёй—®йўҳгҖӮunsafeеҢ…еҸҜд»Ҙи®©зЁӢеәҸе‘ҳз»•иҝҮGoжҳҫејҸиҰҒжұӮзҡ„дҝқжҠӨжңәеҲ¶
  5. Goдёӯзҡ„ж•°з»„жҳҜдёҖзұ»еҖјгҖӮеҪ“дёҖдёӘж•°з»„иў«з”ЁдҪңеҮҪж•°еҸӮж•°ж—¶пјҢеҮҪж•°жҺҘж”¶зҡ„жҳҜж•°з»„зҡ„жӢ·иҙқеҖјпјҢиҖҢдёҚжҳҜжҢҮеҗ‘ж•°з»„зҡ„жҢҮй’ҲгҖӮе®һйҷ…дёҠпјҢеҮҪж•°йҖҡеёёдјҡдҪҝз”ЁslicesдҪңдёәеҸӮж•°пјҢslicesжӢҘжңүжҢҮй’ҲжҢҮеҗ‘ж•°з»„
  6. StringsжҳҜз”ұGoиҜӯиЁҖжҸҗдҫӣзҡ„еҖјзұ»еһӢпјҢдёҖж—ҰеҲӣе»әпјҢдёҚиғҪиў«дҝ®ж”№
  7. GOиҜӯиЁҖдҪҝз”Ёmapsе®һзҺ°е“ҲеёҢиЎЁ
  8. GoиҜӯиЁҖжҸҗдҫӣзәҝзЁӢй—ҙзҡ„йҖҡдҝЎ
  9. maps е’Ң channels жҳҜдј еј•з”ЁпјҢиҖҢдёҚжҳҜдј еҖјгҖӮ
  10. GoиҜӯиЁҖдёҚдҪҝз”ЁеӨҙж–Үд»¶гҖӮGoиҜӯиЁҖзҡ„жҜҸдёӘжәҗж–Үд»¶жҳҜеұһдәҺpackageзҡ„дёҖйғЁеҲҶгҖӮеҪ“еҢ…дёӯе®ҡд№үзҡ„еҜ№иұЎжҳҜд»ҘеӨ§еҶҷеӯ—жҜҚејҖеӨҙзҡ„пјҢиҝҷдёӘеҜ№иұЎеҜ№еҜје…Ҙе®ғзҡ„ж–Үд»¶жҳҜеҸҜи§Ғзҡ„
  11. GoдёҚж”ҜжҢҒйҡҗејҸзұ»еһӢиҪ¬жҚўгҖӮеҢ…еҗ«еӨҡз§Қзұ»еһӢзҡ„ж“ҚдҪңйңҖиҰҒcastпјҲиҪ¬жҚўпјҢеңЁGoдёӯеҸ«conversionпјүгҖӮ
  12. GoдёҚж”ҜжҢҒеҮҪж•°йҮҚиҪҪе’ҢиҮӘе®ҡд№үж“ҚдҪңз¬ҰпјҲoperatorsпјү
  13. Go дёҚж”ҜжҢҒ const жҲ– volatile йҷҗе®ҡз¬Ұ
  14. GoдҪҝз”Ё nil иЎЁзӨәж— ж•Ҳзҡ„жҢҮй’ҲгҖӮC++ дҪҝз”Ё NULL or 0 (or in C++11, nullptr)
  15. Idiomatic Go uses multiple return values to convey errorsвҖ”one or more data results plus an error codeвҖ”instead of sentinel values (e.g., -1) or structured exception handling (C++вҖҷs tryвҖҰcatch and throw or GoвҖҷs panicвҖҰrecover).GoеҸҜд»Ҙиҝ”еӣһеӨҡдёӘеҖј
